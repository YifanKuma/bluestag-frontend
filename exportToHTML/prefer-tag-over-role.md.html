# jsx-a11y/prefer-tag-over-role

<!-- end auto-generated rule header -->

Enforces using semantic DOM elements over the ARIA `role` property.

## Rule details

This rule takes no arguments.

### Succeed

```jsx
<div>...</div>
<header>...</header>
<img alt="" src="image.jpg" />
```

### Fail

```jsx
<div role="checkbox">
<div role="img">
```
